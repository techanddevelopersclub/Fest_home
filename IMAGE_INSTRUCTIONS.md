## Steps to Add Images

1. Open the `public` folder in your project directory:
   ```
   event-management-react-project/public/
   ```

2. Save your images with the exact filenames mentioned above:
   - `director.jpg` (Image 1 - Dr. Sharad K. Pradhan)
   - `dean.jpg` (Image 2 - Mr. Himanshu Nautiyal)
   - `faculty.jpg` (if available)

3. The images will automatically appear on your website

## Image Requirements

- **Format**: JPG/JPEG, PNG recommended
- **Recommended Size**: 400x500 pixels (portrait orientation)
- **File Size**: Keep under 500KB for fast loading

## Verification

After adding the images, refresh your website. If an image doesn't load, check:
- Filename matches exactly (case-sensitive)
- Image is in the `public` folder
- Image file is not corrupted
- Browser cache is cleared (Ctrl+F5)
